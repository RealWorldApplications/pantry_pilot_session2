# Brand Identity Rules
1. Color Palette: Deep Charcoal (#121212), Neon Emerald (#50FFAB), and Pearl White (#F5F5F5).
2. Design Style: Always use Glassmorphism for overlays (Blur: 15.0, Opacity: 0.2).
3. Geometry: All cards and buttons must have a BorderRadius of 24.0.
4. Spacing: Use a strict 8dp grid system for all padding and margins.
